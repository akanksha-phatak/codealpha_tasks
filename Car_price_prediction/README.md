# 🚗 Car Price Prediction using Machine Learning

## 📌 Overview
This project predicts car prices based on features like mileage, fuel type, and year using machine learning.

## 🛠️ Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## ⚙️ Model Used
- Linear Regression
- Random Forest (optional)

## 📊 Results
- Achieved good R² score
- Model predicts car prices with reasonable accuracy

## 📸 Outputs
- Dataset preview
- Model evaluation metrics
- Actual vs Predicted graph

## 🚀 Applications
- Used car price prediction (Cars24, OLX)
- Dealer pricing systems
- AI-based valuation tools
