# 🌸 Iris Classification using Machine Learning

## 📌 Overview
This project focuses on classifying iris flowers into different species using machine learning techniques.

## 📊 Dataset
The dataset contains features such as:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

## 🛠️ Technologies Used
- Python
- Pandas
- Scikit-learn

## ⚙️ Model Used
- Random Forest Classifier

## 📈 Results
- Achieved high accuracy in classifying iris species
- Model successfully predicts flower category based on input features

## 🚀 Applications
- Pattern recognition
- Classification problems
- Educational ML projects
