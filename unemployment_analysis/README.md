# 📊 Unemployment Data Analysis using Python

## 📌 Overview
This project analyzes unemployment trends using data visualization techniques and highlights the impact of Covid-19.

## 📊 Dataset
The dataset includes:
- Date
- Region/State
- Estimated Unemployment Rate (%)

## 🛠️ Technologies Used
- Python
- Pandas
- Matplotlib

## 📈 Analysis Performed
- Trend analysis over time
- State-wise comparison
- Covid-19 impact analysis

## 📊 Results
- Identified a sharp rise in unemployment during Covid-19
- Observed variations across different states
- Noted gradual recovery post-pandemic

## 🚀 Applications
- Economic analysis
- Policy planning
- Data visualization projects
